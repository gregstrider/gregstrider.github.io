PROTEÇÃO DE BAÚS:
Baús, fornalhas e demais itens de interação são PRIVADOS por padrão.
Caso queira permitir que alguém abra seus baús privados, use o comando /trust add [jogador] (sem colchetes).
Para remover, troque o add por remove. Note que esse comando permite acesso do jogador a TODOS os seus baús.

Caso queira liberar apenas alguns baús, use /remover e clique com o botão direito no item para remover a proteção padrão (privado).
Em seguida, use: /cpassword [senha] para trancar o baú com uma senha OU /publico e clique com o botão direito no item para torná-lo público.
O item ficará protegido contra destruição, mas qualquer um poderá acessá-lo (e pegar/colocar itens)!
